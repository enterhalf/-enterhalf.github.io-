SSH秘钥设置成功后, 无需登录Git账号即可推送到Github仓库
由于无需登录, 在Github仓库上显示的提交用户可以是任何人

提交用户的显示优先级分为三级别, System —> Global —>Local; System 即系统级别, 为系统主机名, Global 为配置的全局, Local 为仓库级别, 优先级是 Local > Global > System
